# Linux

## Linux Command Line
- Rename files
  ```bash
  mv file1.txt file2.txt
  ```

- Read files
  ```bash
  cat file1.txt
  ```
  - Can scroll down
  ```bash
  less file1.txt
  ```
  
- Copy contents
  ```bash
  cp file2.txt file1.txt
  ```

- Remove directory
  ```bash
  rmdir dir2
  ```
  - Recursively
  ```bash
  rm -R dir2
  ```
  
- Create a file
  ```bash
  touch file1.txt
  ```

- which version
  ```bash
  which mongod
  ```

- last 500 commands
  ```bash
  history
  ```

- network information 
  ```bash
  ifconfig
  ```
  - wireless 
  ```bash
  iwconfig
  ```

- ping another machine
  ```bash
  ping google.com
  ```
  
- system information
  ```bash
  uname -a
  ```
  
- hard-drive information
  ```bash
  blkid
  ```
  
- processes (like task manager, e.g. memory, CPU)
  ```bash
  top
  ```

- disk space
  ```bash
  df
  ```

- show devices like usb
  ```bash
  lsusb
  ```

- package manager
  ```bash
  sudo apt-get install filezilla
  ```
  ```bash
  sudo apt-get remove filezilla
  ```
  
- shutdown
  ```bash
  sudo shutdown -h now
  ```
  - restart
  ```bash
  sudo shutdown -r
  ```
